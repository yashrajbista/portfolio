VerticalIR: Intelligent Vertical Search & Document Clustering System

VerticalIR is a two-part system that leverages natural language processing and machine learning to enhance academic information retrieval. The project combines a custom-built vertical search engine with a document clustering system to provide fast, topic-aware access to scholarly content. The vertical search engine uses a Python-based web crawler to scrape, extract, and index research publications from Coventry University's School of Computing. An inverted index structure supports efficient search, and a GUI enables intuitive interaction with search results in real time. The second component uses TF-IDF, K-Means, and Decision Trees to automatically cluster and classify documents into topics like Politics, Entertainment, and Economics. This includes full preprocessing (tokenization, lemmatization), real-time classification through a GUI, and performance evaluation using precision, recall, and F1-score. VerticalIR showcases the fusion of web crawling, indexing, unsupervised learning, and user-centric design to deliver intelligent and efficient document discovery and classification.
